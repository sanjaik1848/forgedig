# Practice Page - LeetCode Style

## 🎯 Features

### 1. **Problems Section**
- **Search Functionality** - Search problems by title
- **Language Filter** - C, C++, Python, Java with emoji icons
- **Difficulty Filter** - Easy, Medium, Hard
- **Status Filter** - Solved, Attempted, Todo
- **Problem Cards** with:
  - Status icons (checkmark, circle)
  - Problem number and title
  - Difficulty level with color coding
  - Acceptance rate
  - Tags (Array, Hash Table, etc.)
  - Hover animations

### 2. **Leaderboard Section**
- **Top 10 Rankings** with:
  - Rank badges (🥇🥈🥉 for top 3)
  - User avatars (emojis)
  - Problems solved count
  - Total points
  - Current streak with ⚡ icon
  - Hover effects on each row

### 3. **Stats Dashboard**
- **4 Stat Cards**:
  - Problems Solved
  - Current Streak
  - Global Rank
  - Total Points
- Animated gradient backgrounds
- Hover scale effects

### 4. **Design Features**
- Animated background with particles
- Smooth transitions and animations
- Responsive design (mobile & desktop)
- Color-coded difficulty levels:
  - 🟢 Easy - Green
  - 🟡 Medium - Yellow
  - 🔴 Hard - Red
- Tab navigation between Problems and Leaderboard
